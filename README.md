# daniloribeiro

My bio built with Vue 3, Vite and Tailwind CSS.


## Project Setup

```sh
npm install

yarn

pnpm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev

yarn dev

pnpm dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build

yarn build

pnpm build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint

yarn lint

pnpm lint
```
